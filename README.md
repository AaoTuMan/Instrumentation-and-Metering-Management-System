# Instrumentation-and-Metering-Management-System
仪器仪表仓库管理系统 这是一个基于 Spring Boot 后端和 Vue.js 前端的企业级设备管理系统，专门用于管理企业内部的仪器仪表设备。  📋 项目概述 这个系统是一个完整的设备管理解决方案，支持设备的借用、归还、状态管理等核心功能，适用于企业内部的设备资产管理。
 技术栈
后端技术
Spring Boot 2.5.9 - 主框架
MyBatis - ORM框架
MySQL 8.0 - 数据库
PageHelper - 分页插件
JWT - 身份认证
Hutool - 工具库
Maven - 项目管理
前端技术
Vue 2.6.14 - 前端框架
Vue Router - 路由管理
Element UI - UI组件库
Axios - HTTP客户端
Vue CLI - 脚手架工具

📦 设备管理
设备具有完整的生命周期管理：

设备状态: 可借用、维修中、已报废
上架状态: 已上架、已报废
设备信息: 名称、描述、所属部门、采购时间
借用限制: 支持设备最大借用数量限制
📋 借用管理
完整的设备借用流程：

借用申请: 员工发起设备借用申请
状态跟踪: 待审核、审核通过、已归还
时间管理: 借用时间、归还时间记录
历史记录: 完整的借用历史追踪
📢 公告管理
系统公告发布

🗄️ 数据库设计
